# 💳 Harviclick Payments Website

This is a professional web application for **Harviclick Payments Private Limited**, designed to provide instant credit card cash services in Hyderabad.

---

## 🚀 Live Website

🔗 https://Udaykumarreddyy.github.io/harviclick-payments/

---

## 📌 Project Overview

Harviclick Payments allows users to convert their **credit card limit into instant cash** with secure and transparent processing.

This website includes:

- Instant cash on credit card services
- Credit card to bank transfer
- Payment services (utility, insurance & more)
- Real-time withdrawal calculator
- Customer testimonials
- FAQs with accordion functionality
